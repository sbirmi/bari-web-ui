
# Requirements

* python3
* Flask

# Setup
```
git clone https://github.com/sbirmi/bari-web-ui.git
python3 -m venv bari-web-ui
source bari-web-ui/bin/activate
pip3 install flask
```

# Running the web-server
```
source bari-web-ui/bin/activate
cd bari-web-ui/src
./start_frontend.sh
```
